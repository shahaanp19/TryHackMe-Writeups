# System Fundamentals
---
## Rooms completed on TryHackMe
- Inside a computer system
- Computer types
- Operating systems introduction
- Windows basics
- Web security essentials

---

## Objective
The purpose of these rooms was to develop a foundational understanding of computer systems, operating systems, and how web technologies interact with underlying infrastructure. These skills are essential in ethical hacking, as they provide the knowledge required to analyze, navigate, and exploit target systems.

---

## Operating System Fundamentals

### System Architecture and Functionality
Understanding how operating systems manage hardware, processes, and memory is critical.

Skills developed include:
- Understanding how operating systems handle processes and resources
- Recognizing how applications interact with the system
- Identifying how system components are structured

In ethical hacking, this knowledge is essential for identifying weaknesses within a system and understanding how exploits interact with the operating system.

---

### Windows System Knowledge
Windows is one of the most commonly targeted operating systems in enterprise environments.

Skills developed include:
- Navigating the Windows file system and directory structure
- Understanding user accounts and permissions
- Identifying system configurations and potential misconfigurations

In ethical hacking, these skills are used during enumeration and privilege escalation, where attackers analyze system structure to gain higher levels of access.

---

## Computer System Understanding

### Internal System Components
Understanding how a computer operates internally provides insight into how software and hardware interact.

Skills developed include:
- Understanding key components such as CPU, memory, and storage
- Recognizing how data is processed and stored
- Identifying how systems execute instructions

For ethical hackers, this knowledge helps in understanding how vulnerabilities may arise at a system level and how exploits can impact system performance or behavior.

---

### Types of Computer Systems
Different systems have different roles and configurations.

Skills developed include:
- Understanding differences between personal computers, servers, and embedded systems
- Recognizing how system roles influence security posture
- Identifying potential targets based on system type

In ethical hacking, this helps determine how to approach a target, as different systems require different attack strategies.

---

## Web and System Interaction

### How Websites Operate on Systems
Web applications rely on underlying systems to function.

Skills developed include:
- Understanding how web servers interact with operating systems
- Recognizing how data flows between client, server, and system
- Identifying how web applications are hosted and executed

In ethical hacking, this knowledge is important for identifying how vulnerabilities in web applications can impact the underlying system, potentially leading to full system compromise.

---

## Practical Application to Ethical Hacking

The knowledge gained from these rooms provides a strong foundation for technical security tasks.

These skills enable an ethical hacker to:
- Navigate and analyze target systems during enumeration
- Understand how operating systems can be exploited
- Identify misconfigurations in system settings and permissions
- Recognize how web applications interact with system-level components

This understanding is essential for progressing into more advanced areas such as privilege escalation, system exploitation, and post-exploitation techniques.

---

## Conclusion
These rooms developed essential system-level knowledge required for ethical hacking. Understanding how computers and operating systems function allows for more effective identification of vulnerabilities and supports deeper technical analysis during security assessments.
